# Retail Product Porfitability & Pricing Optimization Analysis

## Project Overview
  This project analyzes retail transaction data to identify opportunities for improving profitability through pricing, discout maanagement, and operational improvements.

  The objective was to approach the dataset from the perspective of a Product Analyst, using SQL and Tableau to investigate the factors influencing profit performance and develop data-driven recommendations that allow for business growth.

## Business Problem
  Revenue growth does not always translate into increased profitability. Excessive discounting, product returns, and underperforming product lines can significantly impact margins and overall business performance.

## Objectives
  The main goals of this project were to to answer the following questions:
  - Which products and categories contribute most to profit?
  - Are discounts helping drive profitable growth?
  - Which products generate revenue but reduce profitability?
  - How do customer segments differ in terms of value?
  - What impact do returns have on financial performance?
  - Are there operation factors affecting profitability?

## Tools Used
  - Microsoft Excel
  - SQL
  - Tableau

## Dataset
  The dataset contains approimately 9,994 retail transactions and includes information such as:
